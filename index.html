Atividade 02: Site HTML Explorando Tags Básicas

1. Tag: < Head>
Função: Titulo ou cabeçalho do site.

2. Tag: < main>
Função: Define o conteúdo principal e é o corpo do documento.

3. Tag: < Tittle>
Função: Define o título da aba no navegador.

4. Tag: < body>
Função: É o conteudo visível na página do site.

5. Tag: < h1> a < h6>
Função: Formatações para títulos do maior(h1) ao menor(h6).

6. Tag: < img>
Função: Serve para inserir imagens.

7. Tag: < hr>
Função: Adiciona uma linha horizontal no site.

8. Tag: < a>
Função: Serve para criar links.

9. Tag: < br >
Função: Realiza uma quebra de linha.
